# STEP1 案件検索

> 出典: ホットプラス会員サイト「入札の流れ」

## 案件の検索方法（cont_id=555）

### ステップ１ タブを【入札検索】に変更する
データウェアネットにログインをすると【マイページ】が表示される。
必ずタブを【入札検索】に変更してから案件検索を行う。

### ステップ2 案件を検索する
案件名のところにキーワードを入れる。
キーワードは、スペースを空けることで複数の単語を一括で検索が可能。

**【注意】**
複数の単語を一括で検索するときは必ず【すべてを含む】を【いずれかを含む】に変更すること。
【すべてを含む】のまま検索をかけてしまうと、案件がヒットしなくなる。

### ステップ3 絞り込み条件を変更する
等級を自身の全省庁統一資格に合わせて変更して検索する。
また、現在入札受付中の案件のみを検索するために【入札受付中】に変更する。

### ステップ4 案件を検索する
案件検索をクリックすると、該当の案件がヒットする。

### その他の機能
- **都道府県毎に案件を検索**: やりたい内容が決まっていないときは、都道府県別で絞り込みが可能
- **発注機関毎に案件を検索**: 気になる発注機関等があれば、発注機関名を入れて検索が可能
- **表示を切り替える**: 都道府県や発注機関名等、表示順を切り替えることが可能
- **CSVに出力する**: CSVに出力してExcel形式で案件を検索することも可能。CSVに出力した場合、地方自治体の案件を省いて検索することも可能

**【注意】**
該当案件が10,000件以上あると表示されない。10,000件以下になるように検索条件を絞り込みを行ったうえで検索すること。
