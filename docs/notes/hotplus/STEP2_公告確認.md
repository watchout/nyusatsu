# STEP2 公告を確認する

> 出典: ホットプラス会員サイト「入札の流れ」

## 参加条件の確認（cont_id=557）

入札の公告には必ず、入札への参加資格に関して記載がされている。
自身の全省庁統一資格で参加が可能か確認すること。

### 確認事項① 全省庁統一資格の記載確認
全省庁統一資格で参加が可能な一般競争入札の場合、必ず【全省庁統一資格】の文言が書かれている。

**注意:**
- 「全省庁統一資格」ではなく、【〇〇に業者登録されている者のうち・・・】や【地方自治法167条の4・・・】等の記載がある場合は、全省庁統一資格では参加できない。

### 確認事項② 営業品目の確認
ほとんどの案件の場合「その他」で参加が可能。
しかし、中には営業品目を求められる場合もあるので、必ず確認を行うこと。

**注意:**
- 営業品目に関して何も書かれていない場合も「その他」で参加が可能
- 営業品目は自身で追加が可能（変更申請が必要）

### 確認事項③ 等級を確認する
全省庁統一資格はA～Dランクまで格付けされている。
ほとんどの案件の場合「D等級」で参加が可能。
しかし、中にはAランクのみやBランクのみ参加が可能な案件もあるので注意が必要。

**重要ポイント:**
- 等級は会社の規模を表す（項目別に点数形式で付けられる）
- 国が決定している等級であり、自身で変更はできない

### 確認事項④ 競争参加地域を確認する
全省庁統一資格は競争参加地域が区切られている。
- ホットプラス経由で取得した場合: 日本全国参加可能
- 自身で取得した場合: 競争参加地域の確認が必要（自身で追加可能）

### その他の参加資格
- **予算決算及び会計令第70条及び71条**: 経営不振状態（自己破産・民事再生）でないこと、暴力団関係者でないことを示す
- **指名停止**: 過去に談合や重大な事故等を起こし、指名停止の処分を受けている場合、参加不可
- **許認可・登録等**: 全省庁統一資格とは別に国家資格が必要な案件の場合に求められる（例: 産業廃棄物の収集運搬の許可証、労働派遣業、建築物環境衛生総合管理業等）
- **実績**: 過去に似たような仕事を行った経験がある場合のみ参加可能な場合がある（例:「一契約あたり、500㎡以上の床清掃作業の履行実績を有する者であること。」）
- **ISO・プライバシーマーク**: 個人情報を扱う案件で、プライバシーマークの取得が条件となる場合がある（個人では取得不可）
- **所在地**: 事業所や本社の所在地を求められる場合がある。事業拠点を証明する書類が必要な場合は官公庁担当者に確認すること
- **物品の製造**: ホットプラス経由で取得した場合は物品の製造は資格取得していない。参加希望の場合は資格取得が必要（製造設備の審査あり）

## スケジュールの確認（cont_id=558）

### 主要な日程項目
- **仕様説明会日時**: 仕事の内容の説明会が開催される日。ほとんどの案件では「なし」。ただし、案件によっては説明会で仕様書を配布される場合もあるので注意
- **入札書提出期限**: 入札書の提出期限。この日時を過ぎて提出された札はすべて無効
- **入札（開札）日時**: 入札書が開かれる日。この日に落札者が決定
- **同等品申請期限**: 物品の案件の場合は見落としがないよう注意
- **履行期限**: 公告の段階で確認可能。必ず確認の上、参加の有無を決定すること

### 参加を希望する場合
入札参加者は「全省庁統一資格」を官公庁に必ず提出する必要がある。
入札に参加できるのは「全省庁統一資格等」を期限内に提出した方のみ。
参加を希望される場合は、余裕をもって参加すること。

## 仕様書の取り寄せ方（cont_id=559）

参加を希望される場合は、仕様書を取り寄せる必要がある。

### 紙入札の場合
電話やFAX等で仕様書を取り寄せる。
連絡先は、公告に必ず記載があるので、担当者を確認の上、発注機関へ連絡する。

### 電子入札の場合
1. 参加したい案件をクリックすると、自動で調達ポータルというサイトに飛ぶ
2. 選択した案件の公告が表示される
3. ページ内から仕様書をダウンロード可能な案件もある

**ダウンロード手順:**
1. ダウンロードボタンをクリック
2. 連絡先情報入力方法選択画面で『連絡先情報をはじめから入力する』にチェック
3. 『次へ』をクリック
4. 利用者情報入力ページで自身の情報を入力
5. 『次へ』をクリック
6. 入力情報の確認画面で確認後『次へ』をクリック
7. ダウンロード画面で『ダウンロード』ボタンをクリック

**注意:** 推奨ブラウザはIE（インターネットエクスプローラー）。お使いのブラウザによってはデータが見れない場合がある。
