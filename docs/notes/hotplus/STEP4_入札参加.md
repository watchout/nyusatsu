# STEP4 入札に参加する

> 出典: ホットプラス会員サイト「入札の流れ」

## 入札書の記入の仕方（cont_id=563）

### 基本ルール
- 入札書・見積書は発注機関側でひな形が準備されている場合がほとんど
- 「機関が定める形式以外での入札は認めません」と言われる場合もある
- 記入漏れ等があると「条件を満たしていないのでこの入札書は無効となります」と言われる場合もある
- **提出前に発注機関へ必ず確認すること**

### 記入方法
- 仕様書に記載がある
- 発注機関によってはホームページ等に記入例や書き方を詳しく明記されている場合もある
- 書き方がわからない場合、必ず発注機関へ確認すること

## 封筒の書き方や提出方法の確認（cont_id=564）

封筒の書き方については、発注機関側から指示がある場合がほとんど。
仕様書等に記載がない場合も念のため、発注機関へ確認すること。

### 提出の方法
封筒は2つ用意する:
- **内封筒**: 入札書（見積書）を入れる
- **外封筒**: 内封筒を入れる

### 内封筒の書き方（指示がない場合の基本形）
封筒サイズ: 長3封筒

**表面:**
1. 発注機関側の担当者名を記入
2. 開札日を和暦で記入
3. 案件名を記入
4. 入札書（見積書）を在中していることを朱書きで記入
5. 自身の情報を記入（全省庁統一資格と相違なく記入）

**裏面:**
- 3か所に割り印

### 外封筒の書き方（指示がない場合の基本形）
- 多くの場合【書留】で送付を指定される
- 事前に連絡をし、許可を得れば、レターパック等でも送付が可能
- 封筒サイズ: 内封筒がおらずにそのまま入るサイズ

**表面:**
1. 発注機関の送付先住所を記入
2. 担当者名を記入
3. 開札日を和暦で記入
4. 案件名を記入
5. 朱書きで【入札書在中】と記入

**裏面:**
6. 自身の情報を記入（全省庁統一資格と相違なく記入）

### レターパックの場合
品名の項目に【案件名】と朱書きで【入札書在中】と記入する。
