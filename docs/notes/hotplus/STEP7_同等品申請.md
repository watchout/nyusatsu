# 同等品申請

> 出典: ホットプラス会員サイト「入札の流れ」

## 同等品申請とは（cont_id=565）

**※同等品申請は入札書を提出する前に行う。同等品で入札される場合は、必ず事前に担当者の承認を得る必要がある。**

### 同等品申請の重要性
- 物品の入札をやるうえで重要なキーポイント
- 同等品申請をやるかやらないかで落札できる可能性がぐっと上がる
- 落札結果を聞いたとき、自分の金額と落札業者の金額が大きく離れている場合は、同等品での入札の可能性が高い

### 同等品申請の定義
官公庁の要求品と同じスペックであれば他社製品でも納入品として認められるというもの。

### 適用範囲
- 物品の販売の案件が主対象
- 役務の案件でも同等品申請は可能（例: エアコンの設置やタイルカーペットの貼り替えの案件で、エアコンやタイルカーペットの製品を同等品申請すると、その分原価を下げることが可能）

## 同等品の探し方（cont_id=637）

### STEP1 同等品申請可能か確認する
- 同等品申請は官公庁によってやり方や書式が異なる
- 仕様書に「同等品」の文字がない場合でも、担当に「同等品の申請は可能ですか？」と確認してみる
- 型番が変わると使用できなくなる場合は、同等品不可と言われることもある
- 仕様書に同等品の文字がなくても、確認すると同等品申請が可能な案件もある

### STEP2 要求品の価格を調べる
- まず要求品の市場最安値を確認する
- 複数メーカーが要求品に指定されている場合は、安い方の価格を基準にする
- 「同じスペックで基準価格以下の製品はないか」を探す

### STEP3 スペックを確認する
- 要求品のスペックをメーカーHPや製品購入ページで確認
- 他製品と比較しやすいようEXCELなどで比較表を作成する（比較表を求められる官公庁もある）

### 具体的に同等品を探す方法

**① 自分で探す**
1. **同じメーカー内で安価なものを探す**: 要求品の旧型品、後継品であっても同等品申請は必要
2. **要求品を検索し、類似品を探す**: 「この商品を見た人は、こんな商品も見ています」等の類似品表示を活用
3. **「安いメーカー名 + 商品カテゴリ」で検索**: YAMAZEN、ツインバード等の安価メーカーで探す

**② 仕入れ業者に同等品での見積を依頼する**
- 「同等品で安価なものがあれば同等品で見積をしてください」とお願いする
- ただし仕入れ業者が仕入れられる範囲内での対応になるので、自身でも調べること

**※同等品申請はいくつでも申請可能。少しでも承認される可能性があるものは申請すべき。**

## 同等品申請の手順（cont_id=638）

### STEP1 同等品申請方法を確認する
- 同等品申請方法は多くの場合、公告に記載されている
- 提出期限までに資料を作成して送る必要がある
- 送付する際の注意点も確認する
- 公告等に記載がない場合は、事前に担当者に申請方法を確認する

### STEP2 同等品申請書類を確認・ダウンロードする
- 同等品申請書は発注機関の方で指定がある場合と、任意の場合がある
- 自衛隊の案件のほとんどはひな形が用意されている
- メールでOKという発注機関や部隊もある
- 申請書類について記載がなければ、事前に確認すること

### STEP3 同等品申請書類を作成する
- ダウンロードした書類に必要事項を記載
- 記載事項は公告や内訳書に記載されている内容を基に該当する部分を記載
- 日付や宛名は記載する場合と空白の場合があり、発注機関によって異なる
- 記載方法でわからない部分があれば、担当者に確認すること

### STEP4 同等品申請書類を提出する
- 公告の提出方法を確認（FAX不可の場合は郵送等）
- 作成した同等品申請書、カタログ、見積もり案内書、内訳書を必要部数用意し提出
- 送り先が不明な場合は、担当者に事前に確認
- 同等品申請の結果が届くのに時間がかかる場合がある
- 見積日までに間に合わない場合は、進捗状況を担当者へ確認すること
