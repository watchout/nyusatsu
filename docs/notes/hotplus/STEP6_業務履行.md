# STEP6 業務を履行する

> 出典: ホットプラス会員サイト「入札の流れ」

## 業務を履行（cont_id=570）

### 物品の販売の場合

**基本フロー:**
1. 配送手配をする
2. お届け日がわかり次第、すぐに機関の担当の方へお届け日を報告する
3. お届け日の翌日になっても担当の方から連絡がない場合は、こちらから連絡し、荷物が届いているか確認する
4. 追跡番号等を使用して、物品の配送状況も併せて確認する

**通販サイトから購入して直接官公庁へ送る場合:**
- お届け先を官公庁にして直接送ることが可能
- 通販サイトのプレゼントやギフト用の設定をすると、価格が表記された明細書を同封することなく送れる
- **重要:** 段ボール等の外箱に配送元として購入元の業者名（amazon等）が記載される。担当の方には前もって「〇〇（amazon）から〇月〇日にお届けします」と必ず伝えること

**会社や自宅等から官公庁へ送る場合:**
- 複数の通販サイトで購入した物品をまとめて送る場合は一度会社や自宅で荷物を受け取り、自身で配達手配をする
- 価格が記載された明細書を同封しないように注意する

**注意:**
- 設置を求められる案件や、納入検査の立会が必要な案件もある
- 機関によっては配送方法や配送業者を細かく指定されている場合がある
- 設置等の記載がなく不安な場合は必ず発注機関へ確認すること

### 役務の提供の場合

- 官公庁の敷地内で作業行う場合は、基本的に作業者名簿を提出する必要がある

**下請け業者に作業を依頼する場合:**
- 下請け業者さんには**落札者業者の人間として**作業に入ってもらう
- 事前に作業者名簿を提出するときも**落札業者の人間として**記入する

## 官側への請求方法（cont_id=571）

### 基本ルール
- 物品の納品や作業終了後、こちらから請求書を官公庁に送らない限り、振込はされない
- **官公庁からの振り込みは基本的に官側が請求書受領後30日以内**
- 業務終了後は速やかに官公庁へ請求書を送ること

### 請求書について
- 基本的に官公庁から雛形がもらえる
- 雛形がなく形式が自由な場合は、インターネットで検索して出てくる請求書のテンプレートで問題ない
